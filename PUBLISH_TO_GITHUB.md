# Publish the Public Try-It Demo (No Command Line)

Recommended repository name: `nexocargo-ai-copilot-demo`

1. Sign in to GitHub.
2. Create a **Public** repository with the name above.
3. Do not initialize it with a README, `.gitignore`, or license.
4. Use GitHub Desktop: **File → Add local repository**, select this folder, then **Publish repository**.
5. On GitHub, open **Settings → Pages**.
6. Under **Source**, choose **Deploy from a branch**.
7. Select branch `main`, folder `/(root)`, then save.
8. Wait for GitHub to display the live Pages URL.
9. Open the URL in a private browser window and test Analyze.
10. Paste that URL into Devpost under the link where judges can try the project.

Expected URL format:

`https://YOUR-GITHUB-USERNAME.github.io/nexocargo-ai-copilot-demo/`
